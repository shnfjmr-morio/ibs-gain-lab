# Changelog

## β1.0.0 — 2026-03-13

### 初回ベータリリース

**食事記録**
- 音声入力・テキスト入力対応
- ローカルFODMAPデータベース（504食品）によるリアルタイムFODMAP判定
- AI分析はオプション（APIキー保持者のみ表示）
- 食後お腹フィードバック（😊/😐/😟）で個人傾向データを蓄積

**FODMAP DB**
- 504食品収録（主食・肉魚・野菜・果物・外食・サプリ等12カテゴリ）
- aliases（表記ゆれ対応）付き
- 日本語・英語の両言語対応

**体重記録**
- 体重・体脂肪率・腸の状態を毎日記録
- 折れ線グラフで推移表示

**AI相談**
- Claude APIキー（BYOK）でAIチャット
- IBS+増量に特化したシステムプロンプト

**設定**
- 日本語/英語切替
- プロフィール・目標カロリー設定
- データエクスポート（JSON）
- **未登録食品ログのエクスポート（CSV）** — DBに未登録だった食品を集計してダウンロード可能

**インフラ**
- PWA（オフライン対応・ホーム画面追加可）
- 全データローカル保存（IndexedDB）
- APIキーはデバイス内のみ保管
